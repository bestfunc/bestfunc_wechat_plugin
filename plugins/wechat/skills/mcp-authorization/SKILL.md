# 微信聊天记录 MCP 授权

本插件所有工具均为 **🟢 只读（L1）**，没有写入/破坏性操作，因此**只需一次 OAuth 登录**，之后直接调用，无需每次验证码审批。

## 授权模型
- **认证(你是谁)**：wxdevops 邮箱验证码登录（OAuth 2.1 授权码 + PKCE）。
- **授权(能看哪些群)**：由系统按你的**邮箱**算可见群——全局管理员可见全部群；群管理员仅可见其负责的群。MCP 返回的数据已自动按此隔离。

## 首次连接流程（标准 OAuth，客户端自动完成大部分）
1. 在 Claude 客户端添加本插件后，首次调用任一 `mcp__wechat__*` 工具时，客户端会打开浏览器跳到授权页。
2. 在授权页用 **wxdevops 邮箱**收验证码登录 → 点「同意授权」。
3. 客户端拿到令牌，后续调用自动带令牌，直接可用。

## 令牌失效 / 401 处理
- 调用返回 401 / `invalid_token` → 令牌过期，重新走一次授权（客户端通常会自动触发；否则重连该 MCP）。
- 看不到某个群 = 你没有该群权限，**不是** bug，不要尝试绕过。

## 给 AI 的提醒
- 不要伪造或猜测令牌/验证码；授权由客户端与用户在浏览器完成。
- 只读工具直接调用即可，无需 `_approval_*` 之类参数。
